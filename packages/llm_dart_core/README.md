# llm_dart_core

Core abstractions and models for the `llm_dart` ecosystem.

This package is provider-agnostic and contains the shared types used across provider implementations (messages, tools, capabilities, configuration, registry, etc.).

## Installation

```bash
dart pub add llm_dart_core
```

## Usage

```dart
import 'package:llm_dart_core/llm_dart_core.dart';
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
