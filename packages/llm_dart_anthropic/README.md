# llm_dart_anthropic

Anthropic provider implementation for the `llm_dart` ecosystem.

Includes Claude chat, reasoning, tools, files, and model listing.

## Installation

```bash
dart pub add llm_dart_anthropic
```

## Usage

```dart
import 'package:llm_dart_anthropic/llm_dart_anthropic.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerAnthropicProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
