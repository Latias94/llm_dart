# llm_dart_xai

xAI (Grok) provider implementation for the `llm_dart` ecosystem.

Includes chat, embeddings, and live search configuration.

## Installation

```bash
dart pub add llm_dart_xai
```

## Usage

```dart
import 'package:llm_dart_xai/llm_dart_xai.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerXAIProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
