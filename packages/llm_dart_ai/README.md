# llm_dart_ai

High-level, provider-agnostic helpers and agents for the `llm_dart` ecosystem (Vercel AI SDK-style), built on top of `llm_dart_core`.

## Installation

```bash
dart pub add llm_dart_ai
```

## Usage

```dart
import 'package:llm_dart_ai/llm_dart_ai.dart';
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
