# llm_dart_openai

OpenAI provider implementation for the `llm_dart` ecosystem.

Includes chat, embeddings, audio, images, files, moderation, and OpenAI Responses API support.

## Installation

```bash
dart pub add llm_dart_openai
```

## Usage

```dart
import 'package:llm_dart_openai/llm_dart_openai.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerOpenAIProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
