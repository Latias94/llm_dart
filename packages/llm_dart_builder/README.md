# llm_dart_builder

Provider-neutral `LLMBuilder` for the `llm_dart` ecosystem.

This package depends only on `llm_dart_core` and provides a fluent API for configuring a provider and building capabilities.

## Installation

```bash
dart pub add llm_dart_builder
```

## Usage

```dart
import 'package:llm_dart_builder/llm_dart_builder.dart';
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
