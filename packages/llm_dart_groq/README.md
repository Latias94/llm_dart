# llm_dart_groq

Groq provider implementation for the `llm_dart` ecosystem.

Built on top of the `llm_dart_openai_compatible` protocol layer.

## Installation

```bash
dart pub add llm_dart_groq
```

## Usage

```dart
import 'package:llm_dart_groq/llm_dart_groq.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerGroqProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
