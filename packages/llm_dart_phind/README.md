# llm_dart_phind

Phind provider implementation for the `llm_dart` ecosystem.

Intended for coding assistant style use cases.

## Installation

```bash
dart pub add llm_dart_phind
```

## Usage

```dart
import 'package:llm_dart_phind/llm_dart_phind.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerPhindProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
