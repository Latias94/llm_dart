# llm_dart_openai_compatible

Reusable OpenAI-compatible protocol implementation for the `llm_dart` ecosystem.

This package provides a shared implementation of the OpenAI REST API shape that can be reused by providers exposing an OpenAI-compatible interface.

## Installation

```bash
dart pub add llm_dart_openai_compatible
```

## Usage

Most users will use a provider package (e.g. `llm_dart_openai`, `llm_dart_groq`) or the `llm_dart` bundle instead of importing this directly.

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
