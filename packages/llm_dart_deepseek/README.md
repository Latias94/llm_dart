# llm_dart_deepseek

DeepSeek provider implementation for the `llm_dart` ecosystem.

Includes chat and model listing with reasoning support.

## Installation

```bash
dart pub add llm_dart_deepseek
```

## Usage

```dart
import 'package:llm_dart_deepseek/llm_dart_deepseek.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerDeepSeekProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
