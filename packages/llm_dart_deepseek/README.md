# llm_dart_deepseek

DeepSeek provider package for `llm_dart`.

## Install

```bash
dart pub add llm_dart_deepseek llm_dart_builder llm_dart_ai
```

## Register provider (subpackage users)

```dart
import 'package:llm_dart_deepseek/llm_dart_deepseek.dart';

void main() {
  registerDeepSeek();
}
```
