# llm_dart_ollama

Ollama provider implementation for the `llm_dart` ecosystem.

Includes chat, completion, embeddings, and model listing for local LLMs.

## Installation

```bash
dart pub add llm_dart_ollama
```

## Usage

```dart
import 'package:llm_dart_ollama/llm_dart_ollama.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerOllamaProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
