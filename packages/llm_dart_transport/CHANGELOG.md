# Changelog

## [0.11.0-alpha.1] - 2026-04-22

- First alpha release of the focused `llm_dart_transport` package.
- Publishes the shared transport abstractions, Dio helpers, SSE/UTF-8 support,
  HTTP chat transport protocol, and transport diagnostics primitives.
- Adds a shared multipart/form-data body builder for provider packages that
  need upload request encoding without duplicating transport utility code.
