# llm_dart_google

Google Gemini provider implementation for the `llm_dart` ecosystem.

Includes chat, embeddings, images, and text-to-speech.

## Installation

```bash
dart pub add llm_dart_google
```

## Usage

```dart
import 'package:llm_dart_google/llm_dart_google.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerGoogleProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
