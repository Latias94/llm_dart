# llm_dart_elevenlabs

ElevenLabs provider implementation for the `llm_dart` ecosystem.

Includes TTS and STT capabilities.

## Installation

```bash
dart pub add llm_dart_elevenlabs
```

## Usage

```dart
import 'package:llm_dart_elevenlabs/llm_dart_elevenlabs.dart';

// Register the provider factory (for llm_dart_core-only setups).
registerElevenLabsProvider();
```

For complete documentation and examples, see the repository:
https://github.com/Latias94/llm_dart
