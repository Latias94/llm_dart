# RBF-020 — Provider Codec Contract Runner

Date: 2026-05-27

## Summary

Implemented `ProviderCodecContractRunner` in `llm_dart_test` as a test-only
provider codec contract runner. It owns fixture lookup, JSON equality,
mismatch diagnostics, and provider stream-event fixture projection.

Concrete request codecs, stream decoders, replay metadata shaping, warning
generation, and provider-native behavior remain provider-owned.

## Deletion Test

Deleting this runner would spread the same fixture lookup and stream-event
projection policy back across OpenAI, Anthropic, Google, and Ollama tests.
The Interface is therefore deep enough to keep.

Error-projection sharing did not pass the deletion test in this slice: the
repo does not yet have repeated provider-owned error golden fixtures. The
generic JSON fixture method can cover those later without introducing an
error-specific shared abstraction now.
