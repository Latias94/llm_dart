# PRF-050 Journal — HTTP Chat Transport Protocol Freeze

Date: 2026-05-27

## Summary

Added a narrow `HttpChatTransportProtocolPolicy` Module and release-facing
protocol policy document.

The policy freezes:

- `uiMessageStreamV2` as the default for new send and reconnect payloads;
- `eventStreamV1` as the fallback for legacy payloads that omit
  `streamProtocol`;
- supported stream protocols as v1 plus v2.

Existing projection, SSE framing, replay state mutation, and stream error
recovery Modules were retained.
